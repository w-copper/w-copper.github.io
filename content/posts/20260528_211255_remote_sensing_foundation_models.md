### 🎯 研究问题

#### 问题背景

遥感图像分割是遥感图像理解的核心任务之一，旨在从高分辨率遥感影像中精确提取感兴趣的地物目标。传统的遥感图像分割方法通常只能处理预定义的类别，无法根据用户的自由文本描述来定位和分割特定目标。

**引用式图像分割 (Referring Image Segmentation)** 是一项新兴任务，要求模型根据自然语言描述（如"建筑物南侧的停车场"、"河流旁边的绿色植被"）来定位并分割图像中对应的目标区域。这项任务在遥感领域具有重要应用价值：

1. **应急响应**：根据文字描述快速定位灾害区域
2. **城市规划**：根据语义描述提取特定用地类型
3. **环境监测**：根据生态特征描述识别植被覆盖

#### 现有方法的局限性

当前的引用式遥感图像分割方法主要存在以下问题：

1. **语义对齐不充分**：现有方法通常使用预训练语言模型编码文本描述，然后与视觉特征对齐，但难以在细粒度语义概念之间建立稳健的对齐关系

2. **表示不一致**：文本和视觉信息之间的表示存在gap，导致模型无法准确理解用户意图

3. **泛化能力不足**：针对特定数据集训练的模型难以泛化到新的场景和描述方式

### 💡 解决方案

#### RSRefSeg 架构设计

RSRefSeg 创新性地整合了两个强大的视觉基础模型：**CLIP** 和 **SAM**，构建了一个统一的引用式遥感图像分割框架。

│   │ (引用描述) │      │    CLIP      │      │   (提示生成)  │        │

#### 核心创新点

**1. 微调CLIP用于遥感场景**

- 采用**低秩自适应 (LoRA)** 技术微调CLIP模型，在保持预训练知识的同时适应遥感领域
- 移除CLIP的池化层，保留原始图像特征图和每个文本token的隐藏状态
- 同时提取全局语义嵌入和局部语义嵌入，实现多粒度特征表示

**2. AttnPrompter 提示生成模块**

AttnPrompter 是连接CLIP和SAM的关键桥梁模块：

- **全局文本语义过滤**：利用全局文本特征作为过滤器，在潜在空间中激活与引用相关的视觉特征
- **局部文本语义增强**：通过局部文本token与视觉特征的注意力机制，增强细粒度语义对齐
- **提示嵌入生成**：将激活的视觉特征转换为SAM可理解的提示嵌入格式

**3. SAM分割能力迁移**

- 利用SAM强大的视觉泛化能力，将CLIP提取的语义提示转化为精确的分割掩码
- 通过提示工程实现零样本或少样本分割能力

#### 技术细节

**低秩微调公式**：

```
W' = W + ΔW = W + BA
```

其中 B ∈ R^(d×r), A ∈ R^(r×k)，r << min(d, k)

**注意力提示生成**：

```python
# 伪代码示意
global_text_feat = clip.encode_text_global(text)  # [1, 512]
local_text_feat = clip.encode_text_local(text)    # [N, 512]
visual_feat = clip.encode_image(image)            # [H, W, 512]

# 全局语义过滤
global_attn = softmax(visual_feat @ global_text_feat.T)
filtered_feat = visual_feat * global_attn

# 局部语义增强
local_attn = softmax(filtered_feat @ local_text_feat.T)
enhanced_feat = filtered_feat + local_attn @ local_text_feat

# 生成SAM提示
prompt_embed = projection(enhanced_feat)
```

## 论文二：LEGNet - 面向低质量遥感图像的轻量级边缘高斯驱动网络

### 📄 论文基本信息

| 项目 | 内容 |
|------|------|
| **论文标题** | LEGNet: Lightweight Edge-Gaussian Driven Network for Low-Quality Remote Sensing Image Object Detection |
| **发表期刊** | IEEE Transactions on Circuits and Systems for Video Technology (TCSVT) 2025 |
| **论文链接** | https://arxiv.org/pdf/2503.14012 |
| **研究方向** | 遥感图像目标检测 (Remote Sensing Object Detection) |
| **核心贡献** | 边缘高斯聚合模块 (EGA) |

### 🎯 研究问题

#### 问题背景

遥感图像目标检测是遥感应用的核心技术之一，广泛应用于城市规划、环境监测、军事侦察等领域。然而，实际遥感图像常面临以下质量问题：

1. **低光照条件**：夜间或云层遮挡导致的图像亮度不足
2. **运动模糊**：卫星或无人机运动造成的图像模糊
3. **遮挡问题**：建筑物、树木等造成的地物目标遮挡
4. **小目标检测**：高分辨率图像中目标尺寸相对较小

#### 现有方法的不足

1. **边缘信息丢失**：传统CNN在特征提取过程中容易丢失高频边缘信息
2. **不确定性处理不足**：对于低质量图像中的模糊区域，缺乏有效的不确定性建模
3. **计算复杂度高**：现有高精度检测模型通常参数量大、推理速度慢

### 💡 解决方案

#### LEGNet 整体架构

#### 核心创新点

**1. LoGStem 边缘增强模块**

LoGStem 是LEGNet的入口模块，专门设计用于增强低质量图像的边缘信息：

- **拉普拉斯高斯滤波 (LoG)**：利用二阶微分算子提取图像边缘特征
- **可学习参数**：将传统滤波器参数化，允许网络自适应学习最优边缘检测核
- **多尺度边缘融合**：在不同尺度上提取边缘信息并进行融合

**2. EGA 边缘高斯聚合模块**

EGA 是LEGNet的核心创新，协同集成了边缘先验和不确定性建模：

**a) Scharr滤波器边缘提取**

```python
# Scharr算子（旋转不变性更好）
Gx = [[-3, 0, 3],
      [-10, 0, 10],
      [-3, 0, 3]]

Gy = [[-3, -10, -3],
      [0, 0, 0],
      [3, 10, 3]]
```

- 面向方向的Scharr滤波器通过旋转不变性保留高频边缘细节
- 相比Sobel算子，Scharr在边缘检测上具有更好的精度

**b) 不确定性感知高斯建模**

```python
# 高斯分布建模
μ = mean(features)  # 均值：特征中心
σ = std(features)   # 标准差：不确定性估计

# 概率特征细化
refined_feat = μ + σ * noise  # 结合不确定性进行特征增强
```

- 通过方差估计概率地优化低置信度特征
- 对模糊区域进行不确定性建模，提高检测鲁棒性

**3. 轻量化设计**

- 采用深度可分离卷积减少参数量
- 使用通道注意力机制提高特征利用效率
- 总参数量仅为传统方法的1/3~1/5
