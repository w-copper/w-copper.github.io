### 2. 研究问题

遥感（Remote Sensing, RS）领域近年来受益于视觉Transformer（ViT）的自监督学习进展，基础模型取得了显著突破。然而，现有方法面临两个核心挑战：

1. **计算复杂度瓶颈**：自注意力机制的二次复杂度（O(n²)）严重限制了模型的可扩展性，尤其在处理大型模型和高分辨率遥感图像时
2. **Mamba应用局限**：虽然线性复杂度的Mamba架构提供了有前景的替代方案，但现有的遥感Mamba应用仅限于小型特定领域数据集上的监督任务，尚未实现大规模自监督预训练

### 3. 解决方案

RoMA框架提出了三个关键创新：

#### 3.1 旋转感知预训练机制（Rotation-Aware Pretraining）
- **自适应裁剪**（Adaptive Cropping）：针对遥感图像中稀疏分布的目标，设计了适应性裁剪策略
- **角度嵌入**（Angular Embeddings）：处理遥感目标任意方向分布的特点，增强模型对旋转不变性的学习能力

#### 3.2 多尺度Token预测目标（Multi-Scale Token Prediction）
- 针对遥感图像中目标尺度变化极大的特点，设计了多尺度预测目标
- 有效捕获不同尺度的语义信息，提升模型对多尺度目标的感知能力

#### 3.3 自回归学习策略（Auto-Regressive Learning）
- 基于Mamba架构的线性复杂度特性，设计了可扩展的自回归预训练框架
- 支持大规模、多样化的无标注遥感数据进行预训练

### 4. 实验结果

#### 4.1 数据集与任务
实验覆盖了三个核心遥感任务：
- **场景分类**（Scene Classification）

#### 4.2 关键发现

1. **Scaling Law验证**：首次在遥感领域验证了Mamba架构遵循数据和参数的缩放定律（Scaling Laws），性能随模型和数据规模的增加而可靠提升

3. **效率提升**：得益于Mamba的线性复杂度，在处理高分辨率遥感图像时展现出显著的计算效率优势

## 论文二：LEGNet - 面向低质量遥感图像的轻量级边缘-高斯网络

### 2. 研究问题

遥感目标检测（RSOD）经常受到以下降质因素的影响：

1. **低空间分辨率**：传感器限制导致图像细节丢失
2. **传感器噪声**：成像过程中的电子噪声干扰
3. **运动模糊**：平台运动或目标运动造成的模糊
4. **不良光照条件**：低光照、阴影等导致的对比度下降

这些因素降低了特征的区分度，导致：
- 目标表示模糊
- 前景-背景分离不充分
- 现有方法在低质量目标检测上鲁棒性不足

### 3. 解决方案

LEGNet提出了轻量级骨干网络，核心是**边缘-高斯聚合模块（EGA）**：

#### 3.1 边缘-高斯聚合模块（Edge-Gaussian Aggregation, EGA）

**组件A：方向感知Scharr滤波器**
- 使用Scharr算子替代传统Sobel算子
- 增强旋转不变性，保留高频边缘细节
- 特别针对低对比度或模糊目标的边缘锐化

**组件B：基于高斯先验的特征细化**
- 通过方差估计实现不确定性感知
- 概率性地优化低置信度特征
- 抑制噪声，增强前景显著性

#### 3.2 轻量级架构设计
- 四阶段网络结构
- 使用带激活函数的归一化层（AN）和卷积层（Conv）
- 在保持计算效率的同时提升特征表示质量

### 4. 实验结果

#### 4.1 评测数据集
在五个基准数据集上进行了全面评估：

| 数据集 | 类型 | 特点 |
|--------|------|------|
| DOTA-v1.0 | 航空影像目标检测 | 大规模、多尺度 |
| DOTA-v1.5 | 航空影像目标检测 | 增加更多小目标 |
| DIOR-R | 遥感目标检测 | 旋转目标 |
| FAIR1M-v1.0 | 高分辨率遥感 | 高精度标注 |
| VisDrone2019 | 无人机视角 | 低空小目标 |

#### 4.2 关键性能

1. **SOTA性能**：在所有五个基准数据集上均达到最先进水平
2. **低质量场景优势**：在遮挡、低光照等复杂条件下展现出稳健的检测能力
3. **效率与精度平衡**：作为轻量级网络，在保持高效率的同时实现了优异的检测精度

### 5. 评价与意义

**技术创新**：
- 将边缘检测（Scharr滤波）与概率建模（高斯先验）创新性结合
- EGA模块设计简洁有效，具有即插即用的潜力

**实用价值**：

**应用前景**：
