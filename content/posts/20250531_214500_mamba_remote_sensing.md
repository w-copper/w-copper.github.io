## 📌 论文一：RoMA —— 遥感领域首个Mamba自监督预训练框架

### 🎯 研究问题：为什么需要Mamba基础模型？

#### 问题背景

遥感（RS）领域近年来见证了Vision Transformer（ViT）自监督学习的突破性进展，催生了一系列遥感基础模型（RSFMs）。然而，**自注意力机制的二次计算复杂度**（O(n²)）成为了可扩展性的重大障碍，尤其是在处理**大规模模型**和**高分辨率遥感影像**时。

#### 核心痛点

1. **计算瓶颈**：ViT处理高分辨率遥感影像时，计算成本呈二次增长
2. **数据规模限制**：现有Mamba在遥感中的应用仅限于小规模、特定领域的监督任务
3. **预训练空白**：缺乏针对Mamba架构的遥感自监督预训练框架
4. **尺度变化极端**：遥感影像中目标尺度变化极大（从微小车辆到大型机场）

#### 作者的切入点

作者注意到Mamba架构具有**线性计算复杂度**（O(n)），是Transformer的有力替代。但现有遥感Mamba研究局限于监督学习小数据集，**没有充分利用大规模无标注遥感数据进行自监督预训练**。这就是RoMA要解决的核心问题。

### 💡 解决方案：旋转感知的多尺度自回归学习

#### 整体框架

RoMA提出了**Rotation-aware Multi-scale Autoregressive learning**框架，首次实现Mamba架构在遥感领域的可扩展自监督预训练。

#### 创新点一：旋转感知预训练机制

**问题洞察**：遥感影像中的目标具有任意方向（如飞机、船舶、建筑物），传统预训练方法忽略了旋转不变性。

**解决方案**：
- **自适应裁剪策略**：根据目标方向自适应裁剪图像区域
- **角度嵌入（Angular Embeddings）**：将旋转角度信息编码到嵌入空间
- **旋转不变表征学习**：在自回归预训练过程中强化旋转不变性

```
传统方法：固定裁剪 → 丢失方向信息 → 表征不鲁棒
RoMA：自适应裁剪 + 角度嵌入 → 保留方向信息 → 旋转鲁棒表征
```

#### 创新点二：多尺度Token预测目标

**问题洞察**：遥感影像中目标尺度变化极端（像素级小目标到区域级大目标），单一尺度预测无法捕捉这种多样性。

**解决方案**：
- 在自回归学习中引入**多尺度预测目标**
- 不同尺度的token预测任务协同训练
- 有效捕捉从小目标到大目标的多尺度特征

#### 技术架构细节

```
输入高分辨率遥感影像
    ↓
自适应旋转裁剪 (Adaptive Rotation-aware Cropping)
    ↓
Mamba Encoder (KV Cache-based Prediction)
    ↓
多尺度Token预测 (Multi-scale Token Prediction)
    ↓
自监督损失计算
```

2. **双重优势**：在精度和计算效率上都具有优势
3. **缩放潜力**：验证了Mamba在遥感领域的缩放潜力

## 📌 论文二：CDMamba —— 将局部线索融入Mamba的变化检测模型

### 📋 论文信息

### 🎯 研究问题：Mamba在变化检测中的局限性

#### 问题背景

遥感变化检测（CD）旨在从不同时期的遥感影像中识别地表变化。近年来，基于状态空间模型（SSM）的Mamba架构因其**线性复杂度**和**长程依赖建模能力**被引入遥感领域。

#### 核心痛点

1. **局部信息缺失**：现有Mamba方法主要关注全局特征提取，**忽略了局部细节信息**
2. **密集预测挑战**：变化检测是密集预测任务，需要精细的像素级判断
3. **双时相交互不足**：现有方法缺乏有效的双时相特征交互机制

#### 作者的切入点

作者发现一个关键问题：**Mamba擅长全局建模，但在密集预测任务（如变化检测）中缺乏局部细节**。这就像用望远镜看全景，却丢失了近处的细节。CDMamba的创新在于**同时捕捉全局和局部信息**。

### 💡 解决方案：全局-局部特征融合

#### 创新点一：Scaled Residual ConvMamba (SRCM) 模块

**设计思想**：将Mamba的全局建模能力与卷积的局部特征提取能力结合。

**架构细节**：
```
输入特征 Fin
    ↓
LayerNorm
    ↓
ConvMamba模块 → 全局+局部特征 Fgl
    ↓
缩放残差连接 (Scaled Residual Connection)
    ↓
LayerNorm + FFN
    ↓
输出特征 Fout
```

**关键创新**：
- ConvMamba模块同时提取全局空间特征（通过Mamba）和局部细节（通过卷积）
- 缩放残差连接增强特征表达能力
- 解决现有Mamba方法"只见森林不见树木"的问题

#### 创新点二：Adaptive Global Local Guided Fusion (AGLGF) 模块

**设计思想**：利用另一个时相的全局/局部特征指导当前时相的特征融合。

**核心机制**：
- **全局引导融合（G-GF）**：利用时相2的全局特征指导时相1的特征融合
- **局部引导融合（L-GF）**：利用时相2的局部特征指导时相1的特征融合
- **自适应权重**：动态调整全局和局部引导的权重

```
时相1特征 + 时相2全局引导 → 增强的变化特征
时相1特征 + 时相2局部引导 → 精细的变化特征
    ↓
自适应融合 → 判别性差异特征
```

**直觉理解**：通过"互相参考"的方式，让两个时相的特征相互指导，从而更准确地识别变化区域。

#### 整体架构

```
时相1影像 ──→ ┐
              ├─→ SRCM Encoder ──→ AGLGF ──→ SRCM Decoder ──→ 变化图
时相2影像 ──→ ┘
```
