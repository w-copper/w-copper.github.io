# 2025年遥感AI前沿：基于基础模型的遥感图像分割最新进展

> 发布时间：2025年5月30日

## 一、论文一：RSRefSeg - 基于基础模型的引用式遥感图像分割

### 1.1 论文信息

| 项目 | 内容 |
|------|------|
| **论文标题** | RSRefSeg: Referring Remote Sensing Image Segmentation with Foundation Models |
| **arXiv编号** | 2501.06809 |
| **发表时间** | 2025年1月 |
| **作者** | Keyan Chen, Chengyang Liu, Wenyuan Li, Hao Chen, Zhenwei Shi |
| **代码仓库** | https://github.com/KyanChen/RSRefSeg |
| **关键词** | 引用式分割、遥感图像、SAM、CLIP、基础模型 |

### 1.2 研究问题

引用式遥感图像分割（Referring Remote Sensing Image Segmentation, RRSIS）是一项新兴的多模态任务，旨在根据自然语言描述从遥感图像中分割出对应的目标区域。该任务面临以下核心挑战：

1. **语义对齐困难**：遥感图像中的目标通常具有复杂的上下文关系，文本描述与视觉特征之间的细粒度语义对齐十分困难
2. **域间差异显著**：通用视觉基础模型（如SAM）在遥感领域的表现受限于训练数据的域间差异
3. **多模态融合挑战**：如何有效融合视觉和文本信息，同时保持各自特征的完整性，是一个关键问题

### 1.3 解决方案

RSRefSeg提出了一种创新的两阶段框架，巧妙地将CLIP和SAM两个基础模型结合：

#### 核心架构

```
输入文本 → CLIP文本编码器 → 全局/局部语义过滤器
                                    ↓
输入图像 → CLIP视觉编码器 → 视觉激活特征 → SAM → 分割掩膜
```

#### 关键创新点

**1. 双重语义过滤机制**
- **全局语义过滤器**：捕获文本描述的整体语义信息
- **局部语义过滤器**：提取细粒度的文本特征，用于精确的目标定位

**2. 视觉激活特征生成**
- 利用CLIP的视觉-语言对齐能力，在潜在空间中生成与引用相关的视觉激活特征
- 这些特征作为提示（prompt）输入到SAM中

**3. SAM的泛化能力利用**
- 通过精心设计的提示工程，充分发挥SAM强大的视觉泛化能力
- 实现从通用域到遥感域的有效迁移

### 1.4 实验结果

#### 数据集
在RRSIS-D数据集上进行了广泛实验，该数据集包含多种遥感场景和复杂的文本描述。

#### 性能对比

| 方法 | oIoU | mIoU | Precision | Recall |
|------|------|------|-----------|--------|
| LAVT | 64.23 | 48.52 | 72.15 | 78.34 |
| ReSTR | 67.89 | 51.23 | 74.56 | 80.12 |
| CRSNet | 69.45 | 53.67 | 75.89 | 81.23 |
| **RSRefSeg** | **73.56** | **58.34** | **79.23** | **84.56** |

RSRefSeg在所有评估指标上均取得了最优性能，显著超越了现有方法。

### 1.5 评价与展望

**优势：**
- 创新性地将CLIP和SAM结合，充分利用了基础模型的能力
- 双重语义过滤机制有效解决了多模态对齐问题
- 代码完全开源，基于MMSegmentation框架，易于复现和扩展

**局限性：**
- 对于复杂场景中多个相似目标的区分能力有待提升
- 推理速度受限于两个大型基础模型的级联使用

## 二、论文二：GSNet - 面向开放词汇的遥感图像语义分割

### 2.1 论文信息

| 项目 | 内容 |
|------|------|
| **论文标题** | Towards Open-Vocabulary Remote Sensing Image Semantic Segmentation |
| **发表会议** | AAAI 2025（CCF-A类会议） |
| **发表时间** | 2025年 |
| **作者** | 大连理工大学研究团队 |
| **代码仓库** | https://github.com/yecy749/GSNet |
| **关键词** | 开放词汇、语义分割、遥感图像、CLIP、LandDiscover50K |

### 2.2 研究问题

传统遥感图像语义分割方法受限于预定义的类别集合，无法处理开放场景中的未知类别。GSNet首次提出了**开放词汇遥感图像语义分割（OVRSISS）**任务，面临以下挑战：

1. **类别封闭性**：现有方法只能识别训练时预定义的类别，无法泛化到新类别
2. **数据集缺乏**：缺少支持开放词汇分割的大规模遥感数据集
3. **域适应困难**：通用视觉语言模型在遥感领域的直接应用效果不佳

### 2.3 解决方案

GSNet提出了一个完整的解决方案，包括新数据集和创新的网络架构：

#### LandDiscover50K数据集

- **规模**：51,846张遥感图像
- **类别**：涵盖40个多样化语义类别
- **来源**：整合了多个知名遥感数据集（Open Earth Map、LoveDA、DeepGlobe等）
- **特点**：统一标注规范，支持开放词汇评估

#### GSNet网络架构

```
                    ┌─────────────────┐
                    │   输入图像      │
                    └────────┬────────┘
                             │
              ┌──────────────┴──────────────┐
              ↓                             ↓
    ┌─────────────────┐           ┌─────────────────┐
    │  CLIP视觉编码器  │           │  RSIB领域模型   │
    │  (通用语义特征)  │           │  (领域特定特征) │
    └────────┬────────┘           └────────┬────────┘
             │                             │
             └──────────┬──────────────────┘
                        ↓
              ┌─────────────────┐
              │  查询引导特征   │
              │  融合(QGFF)    │
              └────────┬────────┘
                       ↓
              ┌─────────────────┐
              │  残差信息保留   │
              │  解码器(RIPD)   │
              └────────┬────────┘
                       ↓
              ┌─────────────────┐
              │   分割结果      │
              └─────────────────┘
```

#### 关键创新点

**1. 双流图像编码器（DSIE）**
- **通用流**：使用CLIP提取通用语义特征，提供开放词汇能力
- **领域流**：使用遥感专用模型RSIB提取领域特定特征
- 两流特征互补，兼顾通用性和专业性

**2. 查询引导特征融合（QGFF）**
- 利用可变词汇表作为查询引导
- 自适应整合通用特征和领域特征
- 避免特征干扰，提升融合效果

**3. 残差信息保留解码器（RIPD）**
- 通过残差连接保留多尺度特征信息
- 增强对小目标和边界区域的分割能力

### 2.4 实验结果

#### 与现有方法对比

| 方法 | mIoU (已知类别) | mIoU (未知类别) | 整体mIoU |
|------|-----------------|-----------------|----------|
| Semantic Seg. | 45.23 | 0.00 | 32.15 |
| OpenSeg | 48.56 | 23.45 | 41.23 |
| SAN | 51.23 | 28.67 | 45.34 |
| **GSNet** | **56.78** | **35.89** | **51.23** |

#### 关键发现

1. **开放词汇能力**：GSNet在未知类别上取得了35.89%的mIoU，显著优于现有方法
2. **已知类别保持**：在已知类别上性能不降反升，证明了双流架构的有效性
3. **泛化性能**：在多个遥感场景下均表现出色

### 2.5 评价与展望

**优势：**
- 首次提出遥感领域的开放词汇分割任务，具有开创性意义
- 构建了大规模数据集LandDiscover50K，为后续研究提供基准
- 双流架构设计巧妙，有效平衡通用性和领域特异性
- 代码和数据集完全开源

**局限性：**
- 对于细粒度类别的区分能力有待提升
- 推理时需要同时运行两个编码器，计算开销较大

## 三、两篇论文的对比分析

### 3.1 研究方向对比

| 维度 | RSRefSeg | GSNet |
|------|----------|-------|
| **任务类型** | 引用式分割 | 开放词汇分割 |
| **输入形式** | 图像 + 文本描述 | 图像 + 任意类别名 |
| **核心模型** | CLIP + SAM | CLIP + RSIB |
| **发表会议** | arXiv预印本 | AAAI 2025 |
| **应用场景** | 根据描述找目标 | 识别任意类别 |

### 3.2 技术路线对比

**共同点：**
- 都利用了CLIP的视觉-语言对齐能力
- 都关注遥感领域的基础模型应用
- 都提供了完整的开源代码

**差异点：**
- RSRefSeg侧重于利用SAM的分割泛化能力
- GSNet侧重于构建领域专用的双流特征提取

### 3.3 应用价值

**RSRefSeg适用于：**
- 灾害响应中根据文字描述快速定位目标
- 军事侦察中根据情报信息提取特定区域
- 城市规划中根据需求描述识别相关设施

**GSNet适用于：**
- 环境监测中识别新出现的地物类型
- 农业遥感中识别新型作物或病害
- 城市管理中发现未定义类别的城市设施

## 四、总结与展望

### 4.1 主要贡献

1. **RSRefSeg**：证明了基础模型在遥感引用式分割中的有效性，为多模态遥感分析提供了新思路
2. **GSNet**：开创了遥感开放词汇分割的新方向，构建了首个大规模基准数据集

### 4.2 未来研究方向

1. **模型轻量化**：如何在保持性能的同时降低基础模型的计算开销
2. **多模态融合**：探索更有效的视觉-语言特征融合机制
3. **时序信息利用**：将时序遥感数据纳入开放词汇分析框架
4. **3D遥感分割**：将开放词汇能力扩展到三维遥感数据

- LandDiscover50K数据集：随GSNet代码一同发布
