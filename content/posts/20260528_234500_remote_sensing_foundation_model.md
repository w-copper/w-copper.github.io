### 2. 研究问题

**核心痛点**：现有遥感基础模型（Remote Sensing Foundation Models）大多仅关注影像数据，忽略了地面级地理空间数据（如OpenStreetMap）所蕴含的丰富地理语义信息。

**具体挑战**：
- **模态鸿沟**：遥感影像与OSM数据在数据结构、内容和空间粒度上存在显著差异
- **协同困难**：如何在预训练和下游任务阶段实现RS与OSM数据的有效融合
- **效率问题**：大规模多模态预训练的计算成本高昂

### 3. 解决方案

GeoLink提出了一个**端到端的多模态框架**，在预训练和下游任务两个阶段同时利用OSM数据增强遥感基础模型：

```
│                    ↓                                     │
│         多粒度学习信号 → 自监督预训练                       │
└─────────────────────────────────────────────────────────┘
```

**关键技术创新**：
- **多粒度学习信号**：从OSM数据中提取多层级地理语义，指导遥感特征学习
- **跨模态空间关联**：利用地理空间位置关系实现RS与OSM信息的交互与协同
- **图像掩码重建**：支持稀疏输入，提升预训练效率

#### 3.2 下游任务适配

GeoLink生成**单模态和多模态细粒度编码**，支持：

## 📚 论文二：GeoPixel —— 遥感影像的像素级多模态理解

### 1. 论文基本信息

| 项目 | 内容 |
|------|------|
| **论文标题** | GeoPixel: Pixel Grounding Large Multimodal Model in Remote Sensing |
| **发表会议** | ICML 2025（国际机器学习会议） |
| **作者团队** | Akashah Shabbir, Mohammed Zumri, Mohammed Bennamoun, Fahad S. Khan, Salman Khan（MBZUAI） |
| **论文链接** | [HuggingFace Papers](https://huggingface.co/papers/2501.13925) |
| **代码仓库** | [GitHub - GeoPixel](https://github.com/mbzuai-oryx/GeoPixel) |
| **数据集** | [GeoPixelD - HuggingFace](https://huggingface.co/MBZUAI/GeoPixelD) |

### 2. 研究问题

**核心痛点**：现有多模态大模型（LMMs）在遥感影像理解中缺乏像素级定位能力，无法实现细粒度的视觉感知与对话。

**具体挑战**：
- **视角差异**：遥感影像的俯视视角与自然图像显著不同
- **尺度变化**：高分辨率遥感影像中目标尺度变化剧烈
- **小目标问题**：遥感场景中存在大量小尺寸目标
- **数据缺乏**：缺少支持像素级遥感对话的标注数据

### 3. 解决方案

GeoPixel是**首个端到端支持像素级定位的高分辨率遥感多模态大模型**：

#### 3.1 核心架构

```
┌──────────────────────────────────────────────────────────────┐
│                    GeoPixel 架构示意                          │
├──────────────────────────────────────────────────────────────┤
│  高分辨率遥感影像（支持4K）                                     │
│           ↓                                                  │
│  [自适应图像分区] → 局部区域 + 全局区域                          │
│           ↓                                                  │
│  [视觉编码器] → 多尺度视觉特征                                 │
│           ↓                                                  │
│  [大语言模型] → 文本生成 + 分割token                            │
│           ↓                                                  │
│  [掩码预测器] → 交错式分割掩码输出                              │
└──────────────────────────────────────────────────────────────┘
```

**关键技术创新**：
- **自适应图像分区**：将高分辨率影像分割为局部和全局区域，支持任意宽高比（最高4K）
- **交错式掩码生成**：在对话中动态生成与文本对应的分割掩码
- **集合标记提示（Set-of-Marks）**：利用空间先验指导数据生成

#### 3.2 GeoPixelD数据集

为支持像素级遥感对话，团队构建了**GeoPixelD**数据集：
- 半自动化标注流程
- 结合场景级上下文与目标级细节
- 支持遥感对话生成（RS-GCG）

### 5. 综合评价

**优势**：
- ✅ 首次实现遥感影像的像素级多模态对话
