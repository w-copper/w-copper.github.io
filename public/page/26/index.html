🔍 研究问题

引用式遥感图像分割（Referring Remote Sensing Image Segmentation, RRSIS） 是遥感领域的一个重要任务，旨在通过自由格式的文本描述来定位并分割遥感图像中的特定目标。

现有方法的主要挑战：

  1. 细粒度语义对齐困难：现有方法通常使用预训练语言模型编码文本描述，并与视觉模态对齐，但难以在细粒度语义概念之间建立稳健的对齐关系
  2. 文本-视觉信息表示不一致：导致跨模态信息传递的误差累积
  3. 跨域迁移性能下降：通用视觉模型在遥感领域的适配存在域差距
  4. 多基础模型知识融合困难：如何有效整合多个基础模型的通用知识是关键挑战

二、研究问题

2.1 背景与挑战

遥感图像分析在环境监测、城市规划、灾害评估等领域具有重要应用价值。近年来，多模态大模型（Large Multimodal Models, LMMs）在自然图像领域取得了显著进展，特别是在细粒度视觉理解和对话生成方面。然而，将这些能力迁移到遥感领域面临以下独特挑战：

  1. 视角差异: 遥感图像采用俯视视角（overhead viewpoint），与自然图像的视角存在本质区别
  2. 尺度变化: 遥感图像中的目标尺度变化范围极大，从小型车辆到大型建筑群
  3. 小目标密集: 高分辨率遥感图像中包含大量小目标，传统LMMs难以有效识别
  4. 数据缺乏: 缺乏支持像素级定位的遥感领域特定数据集

1.2 研究问题

当前遥感基础模型面临三大核心挑战：

（1）传感器模态碎片化 现有基础模型大多针对单一传感器设计（如仅处理光学RGB或多光谱数据），无法处理遥感领域多样化的传感器模态，包括：

  • Sentinel-1 合成孔径雷达（SAR）

（3）评估标准不统一 缺乏覆盖多传感器、多任务的系统性评估基准，导致不同模型之间的比较缺乏公平性。

论文一：GeoPixel - 遥感像素级视觉定位大模型

1. 论文基本信息

项目内容
论文标题GeoPixel: Pixel Grounding Large Multimodal Model in Remote Sensing
作者Akashah Shabbir, Mohammed Zumri, Mohammed Bennamoun, Fahad Shahbaz Khan, Salman Khan
发表会议ICML 2025（国际机器学习会议，CCF-A类）
arXiv2501.13925
代码仓库https://github.com/mbzuai-oryx/GeoPixel
发布时间2025年1月
关键特性首个支持像素级定位的遥感大模型，支持4K分辨率

2. 研究问题

2.1 核心挑战

遥感图像的视觉理解面临独特挑战：

  1. 俯视视角特殊性：遥感图像通常为鸟瞰视角，与自然图像的视角差异巨大，现有大模型难以直接迁移
  2. 尺度变化剧烈：同一场景中目标尺寸差异可达数十倍，小目标检测尤为困难
  3. 缺乏细粒度定位数据：遥感领域缺少支持像素级定位对话的标注数据集
  4. 高分辨率处理困难：遥感图像分辨率常达数千像素，现有模型难以高效处理

2.2 现有方法的局限

  • 现有的大语言模型（LMM）主要针对自然图像设计，在遥感场景中表现不佳
  • 传统遥感分割方法缺乏对话交互能力，无法支持开放式视觉理解
  • 缺乏统一的像素级遥感理解框架

2. 研究问题

  • 任务碎片化：现有方法针对不同任务（分类、检测、分割、定位）使用独立模型，接口不统一

论文一：SkySense-O — 开放世界遥感图像解译的视觉中心视觉语言建模

基本信息

项目内容
论文题目SkySense-O: Towards Open-World Remote Sensing Interpretation with Vision-Centric Visual-Language Modeling
发表会议CVPR 2025
作者团队Qi Zhu, Jiangwei Lao, Deyi Ji, Junwei Luo, Kang Wu, Yingying Zhang, Lixiang Ru, Jian Wang, Jingdong Chen, Ming Yang, Dong Liu, Feng Zhao
论文链接CVPR 2025 Open Access
代码仓库https://github.com/zqcrafts/SkySense-O
模型权重聚合CLIP和SAM版本的SkySense模型

研究问题

开放世界图像解译（Open-World Interpretation）旨在通过视觉语言模型（VLM）准确定位和识别图像中的所有目标物体。尽管在自然图像领域已取得显著进展，但在遥感（RS）图像上的应用仍面临两大核心挑战：

挑战一：语义类别有限

现有遥感语义类别数量有限，尤其是像素级解译数据集更为稀缺。遥感图像覆盖的地物类型极其丰富，从建筑物、道路到农田、水体，再到各种细粒度的土地利用类型，现有数据集难以全面覆盖。

挑战二：空间分布复杂

遥感图像中目标密集、空间分布复杂，仅依靠语言空间难以区分多样的遥感空间区域。与自然图像不同，遥感图像从鸟瞰视角拍摄，目标之间往往紧密相邻，缺乏明显的视觉边界。

解决方案

SkySense-O提出了一种以视觉为中心的视觉语言建模方法，从数据和模型两个层面解决上述挑战。
