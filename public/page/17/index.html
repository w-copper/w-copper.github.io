论文一：RoMA - 基于Mamba的遥感基础模型扩展框架

📄 论文信息

项目内容
论文标题RoMA: Scaling up Mamba-based Foundation Models for Remote Sensing
arXiv链接https://arxiv.org/abs/2503.10392
发表会议NeurIPS 2025（顶级机器学习会议）
GitHubhttps://github.com/MiliLab/RoMA
研究单位国防科技大学、清华大学、武汉大学、北京邮电大学
发表时间2025年

🔍 研究问题

遥感图像具有超高分辨率（通常为数千到数万像素）的特点，这给现有深度学习模型带来了巨大挑战：

  1. 计算复杂度瓶颈：当前主流的Vision Transformer（ViT）基于自注意力机制，其计算复杂度为O(n²)，当处理高分辨率遥感图像时，计算和内存开销急剧增长
  2. 可扩展性受限：由于二次复杂度的限制，现有遥感基础模型难以扩展到更大规模，限制了模型性能的进一步提升
  3. 长距离依赖建模不足：遥感场景中地物的空间分布范围广、尺度变化大，需要模型具备强大的全局上下文建模能力

💡 解决方案

RoMA提出了首个用于遥感的Mamba自监督预训练框架，核心创新包括：

1. 线性复杂度的Mamba架构

  • 采用**状态空间模型（SSM）**替代传统自注意力机制
  • 将计算复杂度从O(n²)降至O(n)，实现线性扩展
  • 保持甚至超越Transformer的全局建模能力

2. 自回归预训练策略

  • 首次将**自回归预训练（Autoregressive Pretraining）**引入Mamba架构用于遥感
  • 设计了适合遥感图像特性的掩码预测任务
  • 在大规模未标注遥感数据上进行自监督学习

3. 多尺度特征融合

  • 设计了层次化的特征提取结构
  • 融合不同尺度的语义信息，适应遥感场景中多尺度地物的检测需求

📊 实验与评估

实验设置

  • 预训练数据：大规模遥感影像数据集
  • 下游任务：场景分类、语义分割、目标检测、变化检测
  • 对比基线：MAE、SatMAE、Scale-MAE等现有遥感预训练方法

主要结果

任务性能提升说明
场景分类+2-3%在多个基准数据集上超越ViT基线
语义分割+1.5-2.5% mIoU在高分辨率分割任务上优势明显
目标检测+1-2% mAP小目标检测提升显著
训练效率3-5x加速相同硬件下训练速度大幅提升

关键发现

  • 首次验证Mamba在遥感领域的Scaling能力：随着模型规模增大，性能持续提升
  • 内存效率显著：处理相同分辨率图像时，内存占用仅为ViT的1/3-1/4
  • 长距离建模优势：在大范围地物（如河流、道路网络）的检测上表现突出

🏆 论文价值

  1. 开创性工作：首个将Mamba架构引入遥感基础模型的大规模研究
  2. 实用性强：代码和预训练模型已完全开源
  3. 影响力大：被NeurIPS 2025接收，代表该领域的最新前沿

一、论文一：GeoPixel —— 遥感领域首个像素级基础大模型

1.1 论文信息

项目内容
标题GeoPixel: Pixel Grounding Large Multimodal Model in Remote Sensing
作者Akashah Shabbir, Mohammed Zumri, Mohammed Bennamoun, Fahad S. Khan, Salman Khan
会议ICML 2025（第42届国际机器学习大会）
机构MBZUAI（穆罕默德·本·扎耶德人工智能大学）
arXiv2501.13925
代码https://github.com/mbzuai-oryx/GeoPixel ⭐ 144
数据集MBZUAI/GeoPixelD

1.2 研究问题

近年来，大型多模态模型（LMMs）在自然图像领域取得了显著进展，能够实现图像理解、视觉问答和视觉定位等任务。然而，当这些模型应用于遥感（Remote Sensing, RS）领域时，面临以下核心挑战：

  1. 视角差异：遥感图像采用俯视视角（overhead viewpoint），与自然图像的平视视角存在根本性差异，导致现有模型的空间理解能力大幅下降。

  2. 尺度变化剧烈：遥感场景中目标尺度跨度极大，从大型建筑到微小车辆，传统LMMs难以有效处理这种多尺度特性。

  3. 小目标密集分布：高分辨率遥感图像中包含大量小目标（如车辆、小型建筑物），这些目标在特征提取过程中容易被忽略。

  4. 缺乏像素级定位数据：现有遥感领域缺乏大规模、细粒度的像素级标注数据集，严重制约了LMMs在遥感场景中的定位对话能力发展。

  5. 分辨率限制：现有模型通常只能处理固定分辨率的输入，无法适应遥感图像分辨率多变的特点。

1.3 解决方案

GeoPixel是首个端到端的高分辨率遥感LMM，支持像素级定位（pixel-level grounding），其核心创新包括：

（1）自适应图像分区（Adaptive Image Divider）

GeoPixel支持最高4K分辨率的任意宽高比输入。通过自适应分区策略，将超大分辨率图像划分为局部区域（local patches）和全局区域（global region），在保持计算效率的同时捕获细粒度空间信息。训练时最大分区数P=9。

（2）双编码器架构

  • 视觉编码器：采用CLIP ViT-L，以560×560分辨率提取视觉特征。
  • 定位视觉编码器：基于SAM2权重初始化，负责生成像素级分割掩码。
  • 语言骨干：使用InternLM-XComposer-2.5（7B参数），通过LoRA（α=8）高效微调。

（3）GeoPixelD数据集构建

一、论文一：Copernicus-FM — 统一的哥白尼基础模型

1.1 论文信息

项目内容
论文标题Towards a Unified Copernicus Foundation Model for Earth Vision
作者Yi Wang, Zhitong Xiong, Chenying Liu, Adam J. Stewart, Thomas Dujardin, Nikolaos Ioannis Bountos, Angelos Zavras, Franziska Gerken 等
发表时间2025年3月（arXiv: 2503.11849）
GitHubhttps://github.com/zhu-xlab/Copernicus-FM
论文链接https://hf.co/papers/2503.11849

1.2 研究问题

当前地球观测（Earth Observation, EO）基础模型存在三大局限性：

  1. 传感器模态受限：大多数基础模型仅针对固定光谱传感器（如Sentinel-2多光谱）设计，无法处理异构传感器数据
  2. 仅关注地表信息：现有模型忽略了大气层等有价值的垂直维度信息
  3. 元数据利用不足：地理坐标、时间戳、传感器参数等丰富元数据在模型中未被充分利用

这些限制导致现有模型在跨传感器迁移、多任务泛化方面表现不佳，难以构建真正通用的地球观测基础模型。

1.3 解决方案

Copernicus-FM 提出了三位一体的创新框架：

（1）Copernicus-Pretrain：大规模多模态预训练数据集

  • 整合了来自哥白尼计划所有主要Sentinel卫星任务的 1870万对齐图像
  • 覆盖范围从地球表面到大气层，跨越光学、SAR、高光谱等多种传感器
  • 数据时间跨度长，包含多时相观测信息

（2）Copernicus-FM：统一基础模型架构

  • 扩展动态超网络（Extended Dynamic Hypernetworks）：根据输入传感器的波段配置动态生成适配权重，实现对任意光谱/非光谱传感器模态的统一处理
  • 灵活元数据编码（Flexible Metadata Encoding）：将地理坐标、时间、传感器参数等元信息融入模型，增强时空感知能力
  • 支持多分辨率、多波段、多传感器的异构数据输入

（3）Copernicus-Bench：系统性评估基准

  • 设计了 15个层次化下游任务，覆盖从预处理到专业应用的完整流程
  • 每个Sentinel卫星任务均有专门的评估基准
  • 支持分类、分割、回归等多种任务类型
