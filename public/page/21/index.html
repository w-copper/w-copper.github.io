一、GeoPixel：像素级遥感大模型（ICML 2025）

1.1 论文信息

项目内容
论文标题GeoPixel: Pixel Grounding Large Multimodal Model in Remote Sensing
发表会议ICML 2025（第42届国际机器学习会议）

GitHubhttps://github.com/mbzuai-oryx/geopixel （⭐ 144）
arXivhttps://arxiv.org/abs/2501.13925
关键词大规模多模态模型、遥感、像素级定位、视觉语言模型

1.2 研究问题

近年来，大规模多模态模型（Large Multimodal Models, LMMs）在自然图像领域取得了显著进展，尤其在细粒度视觉定位（grounding）方面表现出色。然而，这些能力在遥感（Remote Sensing, RS）领域面临严峻挑战：

  1. 视角差异：遥感图像采用俯视视角，与自然图像的平视视角存在本质区别
  2. 尺度变化：遥感图像中目标尺度变化剧烈，从小型车辆到大型建筑物跨度极大
  3. 小目标密集：高分辨率遥感图像中包含大量小目标，增加了检测和分割难度
  4. 数据匮乏：缺乏带有像素级标注的遥感对话数据集，限制了定位对话能力的发展

1.3 解决方案

GeoPixel是首个支持像素级定位的端到端高分辨率遥感大模型，具有以下核心创新：

（1）自适应图像分区

GeoPixel采用创新的自适应分区策略，将输入图像动态划分为局部（local）和全局（global）区域：

  • 局部区域：捕获细粒度细节信息
  • 全局区域：保持整体场景理解
  • 支持4K超高清分辨率，适用于任意宽高比

（2）半自动数据生成管道

为解决遥感领域像素级标注数据匮乏的问题，研究团队构建了GeoPixelD数据集：

  • 采用半自动化的数据生成流程
  • 利用Set-of-Marks提示和空间先验
  • 系统性控制数据生成质量
  • 生成带有交错掩码（interleaved masks）的对话数据

论文一：Copernicus-FM — 统一的哥白尼地球观测基础模型

📄 论文信息

项目内容
标题Towards a Unified Copernicus Foundation Model for Earth Vision
作者Yi Wang, Zhitong Xiong, Chenying Liu, Adam J. Stewart, Thomas Dujardin 等
会议ICCV 2025 Oral（国际计算机视觉大会，口头报告）
论文arXiv:2503.11849
代码github.com/zhu-xlab/Copernicus-FM（⭐ 133 Stars）
关键词基础模型、多模态融合、地球观测、自监督学习

🔍 研究问题

当前地球观测（Earth Observation, EO）基础模型面临三大核心瓶颈：

  1. 传感器模态割裂：现有模型通常只能处理单一或固定的传感器组合（如仅光学或仅SAR），无法适应真实世界中多种传感器并存的场景
  2. 数据覆盖局限：大多数预训练数据仅关注地表信息，忽略了大气层、海洋等多层次地球观测数据
  3. 元数据利用不足：卫星图像附带的丰富元数据（如地理坐标、拍摄角度、大气条件）在现有模型中几乎未被有效利用

这些问题导致模型的泛化能力受限，难以应对实际应用中多样化、异构化的地球观测需求。

💡 解决方案

Copernicus-FM提出了三位一体的创新框架：

1. Copernicus-Pretrain：大规模多模态预训练数据集

  • 规模：整合了1870万张对齐图像，来自所有主要哥白尼哨兵卫星任务（Sentinel-1至Sentinel-5P）
  • 覆盖范围：从地表到大气层的全方位地球观测数据
  • 数据特点：多传感器、多时相、多分辨率的全球采样数据

2. Copernicus-FM：统一基础模型架构

  • 动态超网络：采用扩展的动态超网络（Extended Dynamic Hypernetworks），能够根据输入传感器的特性动态调整模型参数，实现对任意光谱或非光谱传感器模态的统一处理
  • 灵活元数据编码：将地理坐标、时间戳、传感器参数等元数据嵌入模型，增强上下文感知能力
  • 掩码图像建模：基于自监督学习范式，通过掩码图像重建任务学习通用视觉表征
  • 持续蒸馏：支持模型的持续更新和优化

3. Copernicus-Bench：系统化评估基准

  • 任务层次：涵盖15个层次化的下游任务，从预处理到各哨兵卫星的专门应用
  • 评估维度：包括分类、分割、回归等多种任务类型
  • 标准化：提供统一的评估协议和基准线

本文精选了2025年发表于ICML和NeurIPS两大顶级会议的遥感AI论文，它们均提供了开源代码，代表了当前遥感智能解译的最新进展。

一、论文一：GeoPixel —— 支持像素级定位的遥感大型多模态模型

1.1 论文信息

  • 标题: GeoPixel: Pixel Grounding Large Multimodal Model in Remote Sensing
  • 会议: ICML 2025 (第42届国际机器学习会议)
  • 作者: Akashah Shabbir, Ilmuz Zaman Mohammed Zumri, Mohammed Bennamoun, Fahad Khan, Salman Khan
  • arXiv: https://arxiv.org/abs/2501.13925
  • 代码: https://github.com/mbzuai-oryx/GeoPixel
  • 主页: https://mbzuai-oryx.github.io/GeoPixel/

1.2 要解决的问题

近年来，大型多模态模型（LMMs）在自然图像领域取得了显著进展，能够实现细粒度的视觉定位和对话。然而，这些模型在遥感（RS）领域的应用效果有限，主要原因在于：

  1. 视角独特：遥感图像通常为俯视视角，与自然图像的视角差异巨大。
  2. 尺度变化大：遥感图像中的目标尺度变化范围广，从小型车辆到大型机场。
  3. 小目标密集：高分辨率遥感图像中包含大量小目标，增加了检测和分割难度。
  4. 数据缺乏：缺乏细粒度的、带有像素级标注的遥感领域专用数据集，限制了模型的训练和评估。

1.3 解决方案

为解决上述问题，作者提出了GeoPixel，这是第一个支持像素级定位的端到端高分辨率遥感大型多模态模型。其核心创新包括：

  1. 高分辨率支持：GeoPixel支持高达4K（3840×2160）分辨率的图像输入，且不限制宽高比，非常适合高精度遥感图像分析。
  2. 自适应图像分区：模型能够动态地将输入图像划分为局部和全局区域，确保高效的编码和分析。
  3. 像素级定位能力：通过在对话中生成交错的分割掩码，实现细粒度的视觉感知。
  4. GeoPixelD数据集：作者构建了一个半自动化的数据生成流程，利用“集合标记”（Set-of-Marks）提示和针对遥感数据设计的空间先验，系统地生成了高质量的定位对话数据集GeoPixelD。

1.4 实验与结果

GeoPixel在多个基准测试中表现出色：

  • 单目标分割：在单目标分割任务中，GeoPixel显著优于现有的通用LMMs。
  • 多目标分割：在更具挑战性的多目标分割任务中，GeoPixel同样展现出卓越的性能，能够准确识别和分割图像中的多个不同类别目标。
  • 消融研究：论文中的方法论消融研究验证了模型各个组件（如自适应分区、空间先验等）的有效性。

1.5 评价与意义

GeoPixel的提出填补了遥感领域大型多模态模型在像素级定位能力上的空白。它不仅支持超高分辨率图像处理，还通过创新的半自动数据生成流程解决了数据稀缺问题。该模型为遥感图像的精细解译、人机交互式分析以及下游应用（如灾害评估、城市规划）提供了强大的工具。
