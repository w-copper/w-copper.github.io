## 一、论文一：RSRefSeg - 基于基础模型的引用式遥感图像分割

### 1.1 论文信息

| 项目 | 内容 |
|------|------|
| **标题** | RSRefSeg: Referring Remote Sensing Image Segmentation with Foundation Models |
| **作者** | Keyan Chen 等 |

| **领域** | 遥感图像分割、多模态学习、基础模型 |

### 1.2 研究问题

**引用式遥感图像分割（Referring Remote Sensing Image Segmentation, RRSIS）** 是指根据自然语言描述在遥感图像中定位并分割出对应的目标区域。该任务面临以下核心挑战：

1. **细粒度语义对齐困难**：现有方法难以在细粒度语义概念之间建立稳健的对齐关系，导致文本和视觉信息之间的表示不一致。

2. **遥感场景复杂性**：遥感图像具有视角多样、目标尺度变化大、背景复杂等特点，传统的视觉-语言模型难以直接迁移应用。

3. **泛化能力不足**：现有方法在处理未见过的类别或场景时，分割性能显著下降。

### 1.3 解决方案

RSRefSeg 提出了一种基于基础模型的创新架构，核心思想是**利用CLIP进行视觉-文本编码，结合SAM进行精细分割**：

#### 整体架构

```
输入文本 → CLIP文本编码器 → 全局/局部语义过滤器
                                    ↓
输入图像 → CLIP视觉编码器 → 引用相关视觉激活特征
                                    ↓
                              SAM分割网络 → 分割掩膜
```

#### 核心创新

1. **双层级语义过滤机制**
   - **全局语义**：捕获整体场景信息
   - **局部语义**：关注细粒度目标特征
   - 通过语义过滤器在潜在空间中生成与引用相关的视觉激活特征

2. **基础模型协同**
   - **CLIP**：负责视觉和文本的对齐编码
   - **SAM（Segment Anything Model）**：利用其强大的视觉泛化能力进行精细分割

3. **激活特征引导**
   - 将CLIP生成的激活特征作为提示（prompt）输入SAM
   - 实现从粗粒度语义理解到细粒度像素分割的端到端流程

### 1.4 实验评估

#### 数据集

- **RRSIS-D**：包含4,420个图像-语言-标签三元组
- **RefSegRS**：大规模遥感引用分割数据集

#### 性能对比

| 方法 | IoU | Precision | Recall |
|------|-----|-----------|--------|
| LAVT | 45.2 | 58.3 | 62.1 |
| ReSTR | 48.7 | 61.2 | 65.4 |
| **RSRefSeg** | **53.6** | **67.8** | **71.2** |

#### 关键发现

- 相比现有SOTA方法，IoU提升约5-10%
- 在复杂场景（多目标、小目标）中表现尤为突出
- 零样本泛化能力显著优于传统方法

## 二、论文二：LEGNet - 轻量级边缘-高斯驱动遥感目标检测网络

### 2.2 研究问题

低质量遥感图像目标检测面临的挑战：

1. **图像质量退化**：航空和卫星图像常受到空间分辨率低、传感器噪声、模糊、低光照等因素影响。

2. **计算资源受限**：边缘设备（无人机、星载处理器）计算能力有限，难以部署大型检测模型。

3. **遮挡与复杂背景**：遥感场景中目标常被树木、建筑物遮挡，背景干扰严重。

### 2.3 解决方案

LEGNet 提出了一种**边缘-高斯协同**的轻量化检测架构：

#### 核心模块：EGA（Edge-Gaussian Aggregation）

```
输入特征图
    ↓
┌───────────────────────────────────────┐
│  Scharr边缘检测 → 方向感知边缘特征    │
│              ↓                        │
│  高斯建模 → 不确定性感知特征细化      │
└───────────────────────────────────────┘
    ↓
增强特征图
```

#### 技术创新

1. **Scharr算子边缘先验**
   - 使用旋转不变的Scharr滤波器保留高频边缘细节
   - 相比Sobel算子，边缘检测更精确

2. **不确定性感知高斯建模**
   - 通过方差估计概率性地优化低置信度特征
   - 对噪声和模糊具有更强的鲁棒性

3. **轻量化设计**
   - 深度可分离卷积减少参数量
   - 通道剪枝实现网络瘦身
