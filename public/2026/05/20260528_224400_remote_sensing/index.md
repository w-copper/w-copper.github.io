## 论文一：RSRefSeg - 基于基础模型的引用式遥感图像分割

### 📄 论文信息

### 🎯 研究问题

引用式遥感图像分割（Referring Remote Sensing Image Segmentation）是一项通过自由格式文本描述来定位和分割遥感图像中特定目标的任务。该任务在遥感图像解译中具有重要应用价值，能够实现细粒度的视觉理解和场景提取。

**现有方法的痛点：**
1. 现有研究通常使用预训练语言模型编码文本描述，并与视觉模态对齐
2. 难以在细粒度语义概念之间建立稳健的对齐关系
3. 文本和视觉信息之间的表示存在不一致性

### 💡 解决方案

RSRefSeg 提出了一个基于基础模型的引用式遥感图像分割框架，核心创新包括：

**1. 双编码器架构**
- 利用 **CLIP** 进行视觉和文本编码
- 采用全局和局部文本语义作为过滤器

**2. 视觉激活特征生成**
- 在潜在空间中生成与引用相关的视觉激活特征
- 这些激活特征作为输入提示用于 SAM

**3. SAM 分割优化**
- 利用 SAM（Segment Anything Model）强大的视觉泛化能力
- 通过激活特征引导 SAM 生成精确的分割掩膜

**技术架构流程：**
```
文本描述 → CLIP文本编码器 → 全局/局部语义特征
                                    ↓
遥感图像 → CLIP视觉编码器 → 视觉激活特征 → SAM → 分割掩膜
```

### 📊 实验与评估

**数据集：** 遥感引用分割基准数据集

**性能表现：**
- 在多个遥感引用分割基准上取得领先性能
- 1.2B 参数规模确保了模型的表达能力
- CLIP + SAM 的组合实现了跨模态的强对齐

**可视化结果：**
- 模型能够准确理解复杂文本描述
- 对不同大小、形状的目标均能精确分割
- 在复杂遥感场景中展现出良好的泛化能力

### 🏆 评价与贡献

1. **首次将 CLIP 和 SAM 结合**应用于遥感引用分割任务
2. **提出全局-局部语义过滤机制**，增强了细粒度语义对齐
3. **代码开源**，便于学术界复现和改进
4. 为遥感基础模型的应用提供了新范式

## 论文二：LEGNet - 面向低质量遥感图像的轻量级边缘高斯驱动网络

### 📄 论文信息

| **作者** | Wei Lu, Si-Bao Chen, Hui-Dong Li, Qing-Ling Shu, Chris H. Q. Ding, Jin Tang, Bin Luo |
| **发表期刊** | TCSVT 2025 / ICCVW 2025 |
| **代码仓库** | https://github.com/lwCVer/LEGNet |
| **论文链接** | https://arxiv.org/abs/2503.14012 |
| **单位** | 安徽大学、香港中文大学（深圳） |

### 🎯 研究问题

遥感目标检测（RSOD）在复杂视觉环境中面临巨大挑战。航空和卫星图像固有地受到以下因素限制：

**核心问题：**
1. **空间分辨率低**：难以识别小目标
2. **传感器噪声**：影响特征提取质量
3. **模糊目标**：边界不清晰
4. **低光降解**：对比度降低
5. **部分遮挡**：目标被建筑物或树木遮挡

**导致的关键问题：**
- 降低的对比度阻碍了前景-背景分离
- 边缘表示中存在结构不连续
- 光照变化引起的模糊特征响应

### 💡 解决方案

LEGNet 提出了专门设计用于低质量遥感图像的轻量级网络，核心创新是 **边缘高斯聚合（EGA）模块**：

**1. Scharr 算子边缘先验**
- 使用面向方向的 Scharr 滤波器
- 通过旋转不变性保留高频边缘细节
- 比传统 Sobel 算子具有更好的边缘检测精度

**2. 不确定性感知高斯建模**
- 高斯层通过方差估计概率地优化低置信度特征
- 对噪声和模糊具有更强的鲁棒性

**3. 轻量化设计**
- 确保计算效率
- 适合部署在资源受限的边缘设备上
- 适用于实际的遥感应用场景

**EGA 模块结构：**
```
输入特征 → Scharr边缘检测 → 边缘先验特征
                              ↓
输入特征 → 高斯不确定性建模 → 概率特征
                              ↓
                    特征聚合 → 增强特征输出
```

### 📊 实验与评估

**基准数据集：**

|--------|------|------|
| DOTA-v1.0 | 航空影像 | 15个类别，大规模 |
| DOTA-v1.5 | 航空影像 | 更多小目标 |
| DIOR-R | 遥感影像 | 旋转目标检测 |
| FAIR1M-v1.0 | 高分辨率 | 精细目标检测 |
| VisDrone2019 | 无人机视角 | 低空拍摄 |

**性能表现：**
- 在 5 个基准数据集上达到最先进的性能
- 在遮挡和低光照等复杂条件下展现出稳健的检测能力
- 被树木或建筑物遮挡的目标，准确性与鲁棒性均超越以往方法

**可视化结果：**
- 在 DOTA-v1.0 测试集上的可视化结果表明
- 所有模型均基于 Oriented R-CNN 构建
- LEGNet 在复杂条件下展现出更强的检测能力

### 🏆 评价与贡献

1. **提出 EGA 模块**：创新性地将 Scharr 边缘先验与高斯不确定性建模结合
2. **轻量化设计**：适合边缘设备部署，具有实际应用价值
