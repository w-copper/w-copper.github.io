论文一：RemoteSAM — 面向地球观测的分割一切基础模型

🎯 研究问题

遥感图像的视觉感知和理解任务（如目标检测、语义分割、视觉定位等）在环境监测、城市规划、灾害评估等领域具有重要应用价值。然而，当前遥感视觉基础模型面临以下核心挑战：

  1. 任务专用性限制：现有模型通常针对单一任务设计，缺乏统一的多任务处理能力
  2. 数据覆盖不足：训练数据的语义类别覆盖范围有限，难以泛化到多样化场景
  3. 输入输出接口不灵活：不同任务需要不同的输入输出格式，缺乏统一的任务接口
  4. 标注成本高昂：遥感图像的像素级标注需要专业知识和大量人力

💡 解决方案

RemoteSAM 提出了两项核心创新：

1. 自动化数据引擎（Automatic Data Engine）

传统方法：人工标注 → 成本高、扩展性差
RemoteSAM：自动数据引擎 → 可扩展性强、语义覆盖广
  • 相比传统人工标注或基于规则的方法，具有显著更好的可扩展性
  • 构建了迄今最大的遥感分割数据集：RemoteSAM-270K
    • 包含 270K 图像-文本-掩码三元组
    • 覆盖前所未有的多样化语义类别和属性规范
    • 数据质量和多样性远超现有数据集

2. 任务统一范式（Task Unification Paradigm）

以**指代表达分割（Referring Expression Segmentation）**为核心，统一处理多种视觉感知任务：

┌─────────────────────────────────────────────────────────┐
│                    RemoteSAM 统一框架                      │
├─────────────────────────────────────────────────────────┤
│  输入：图像 + 文本描述                                       │
│  输出：分割掩码 / 边界框 / 类别标签                            │
├─────────────────────────────────────────────────────────┤
│  支持任务：                                                │
│    ✓ 图像分类 (Classification)                             │
│    ✓ 目标检测 (Detection)                                  │
│    ✓ 语义分割 (Segmentation)                               │
│    ✓ 视觉定位 (Grounding)                                  │
│    ✓ 指代分割 (Referring Segmentation)                     │
└─────────────────────────────────────────────────────────┘
  • 无需任务特定头部：单一模型处理所有任务，无需为每个任务设计专用网络头
  • 文本驱动：通过自然语言描述指定目标，实现灵活的视觉感知

论文二：RSRefSeg — 基于基础模型的指代遥感图像分割

📄 论文信息

项目内容

发表时间2025年1月
作者Keyan Chen 等
GitHubhttps://github.com/KyanChen/RSRefSeg
模型参数约1.2B
论文链接https://arxiv.org/abs/2501.06809

🎯 研究问题

指代遥感图像分割（Referring Remote Sensing Image Segmentation）是一项通过自由格式文本输入实现细粒度视觉理解的关键任务。现有方法面临以下挑战：

  1. 语义对齐困难：文本描述与视觉特征之间难以建立鲁棒的细粒度语义对齐
  2. 表示不一致：文本和视觉信息的表示存在不一致问题
  3. 领域适应性差：通用视觉-语言模型在遥感领域的表现受限
  4. 空间关系理解不足：遥感图像中的空间关系复杂，现有方法难以准确捕捉

💡 解决方案

RSRefSeg 提出了一种基于CLIP和SAM的遥感指代分割基础模型：

核心架构

┌─────────────────────────────────────────────────────────┐
│                     RSRefSeg 架构                         │
├─────────────────────────────────────────────────────────┤
│  视觉编码器：CLIP ViT                                      │
│    ↓                                                      │
│  文本编码器：CLIP Text Encoder                              │
│    ↓                                                      │
│  特征融合：全局 + 局部文本特征                                │
│    ↓                                                      │
│  分割解码器：SAM Decoder                                    │
│    ↓                                                      │
│  输出：像素级分割掩码                                        │
└─────────────────────────────────────────────────────────┘

关键技术创新

  1. 双层文本特征提取

    • 全局特征：捕捉文本描述的整体语义
    • 局部特征：捕捉细粒度的目标属性和空间关系

  2. CLIP驱动的视觉-文本编码

    • 利用CLIP强大的视觉-语言对齐能力
    • 预训练的CLIP模型提供良好的跨模态表示基础

  3. SAM分割解码器

    • 借助SAM强大的分割能力
    • 实现高质量的像素级分割
