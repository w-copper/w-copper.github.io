### 2. 研究问题

遥感变化检测（Change Detection）是通过比较同一区域不同时期的遥感影像，识别地表变化的关键技术。该任务面临以下核心挑战：

**传统方法的局限性：**
- **CNN方法**：感受野有限，难以捕捉长距离时空依赖关系
- **Transformer方法**：自注意力机制计算复杂度为O(n²)，处理高分辨率遥感影像时显存消耗巨大
- **多时相融合**：现有方法在融合双时相特征时，难以有效建模时空交互关系

**核心痛点：**
如何在保持线性计算复杂度的同时，有效捕捉遥感影像中的长距离时空依赖关系？

### 3. 解决方案

ChangeMamba首次将**状态空间模型（State Space Model, SSM）**引入遥感变化检测领域，提出了三种变体架构：

#### 3.1 核心创新：MambaBCD（二值变化检测）

```
架构设计：
┌─────────────────────────────────────────────────────────┐
│                    MambaBCD架构                          │
├─────────────────────────────────────────────────────────┤
│  双时相输入 → Siamese Encoder → Temporal Mamba Block →   │
│  → Difference Enhancement → Decoder → 变化检测图         │
└─────────────────────────────────────────────────────────┘
```

**关键技术：**
- **时空状态空间块（Spatio-Temporal SSM Block）**：利用Mamba的选择性扫描机制，实现线性复杂度的长序列建模
- **双向状态空间模型**：从前向和后向两个方向捕捉时空依赖关系
- **差异增强模块**：强化双时相特征的差异表达

#### 3.2 MambaSCD（语义变化检测）

在MambaBCD基础上增加语义分类头，同时检测变化区域和变化类型（如：从"农田"变为"建筑"）。

#### 3.3 MambaBDA（建筑损害评估）

针对灾后建筑损害评估场景，实现像素级的损害等级分类。

### 2. 研究问题

**引用式遥感图像分割（Referring Remote Sensing Image Segmentation, RRSIS）**是一项新兴任务：给定一张遥感图像和一段自然语言描述（如"图像左上角的红色建筑"），模型需要分割出对应的区域。

**现有方法的挑战：**
- **语义对齐困难**：细粒度语义概念（如"较大的"、"靠近河流的"）难以与视觉特征精确对齐
- **泛化能力不足**：针对特定场景训练的模型难以泛化到新的地理区域或传感器
- **基础模型适配**：如何将通用视觉基础模型（CLIP、SAM）有效适配到遥感领域

### 3. 解决方案

RSRefSeg创新性地整合了**CLIP**和**SAM**两大基础模型：

#### 3.1 整体架构

```
│  │  描述   │    │    CLIP      │    │ Prompter│      掩码    │
│  └─────────┘    └──────────────┘    └─────────┘             │
│                       ↑                                      │
│  ┌─────────┐         │                                      │

└──────────────────────────────────────────────────────────────┘
```

#### 3.2 核心创新

**1. 低秩微调CLIP（Fine-tuned CLIP）**
- 引入LoRA（Low-Rank Adaptation）高效微调策略
- 移除CLIP的池化层，保留原始图像特征图和文本token隐藏状态
- 实现遥感领域的细粒度语义对齐

**2. 注意力提示器（AttnPrompter）**
- 作为CLIP和SAM之间的桥梁
- 将CLIP提取的引用语义信息转换为SAM可理解的提示嵌入
- 支持全局和局部文本语义的融合

**3. 微调SAM（Fine-tuned SAM）**
- 利用SAM强大的视觉泛化能力
- 根据提示嵌入处理原始图像，生成精确的二值分割掩码
