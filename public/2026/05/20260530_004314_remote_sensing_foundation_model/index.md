### 🎯 研究问题

1. **数据异质性**：遥感数据来自不同传感器模态，具有不同的空间和时间维度
2. **标注数据稀缺**：高质量的遥感标注数据获取成本高昂
3. **多尺度特征需求**：地球表面现象具有不同的尺度和类型，需要模型能够同时捕捉大尺度和小尺度特征
4. **通用性不足**：现有模型通常针对特定任务设计，难以跨任务迁移

- 能够处理不同形状的数据（不同的空间和时间维度）

#### 2. 新型自监督学习方法
- 提出了一种**新颖且高效的自监督学习方法**
- 能够同时学习大尺度和小尺度特征
- 解决了先前模型未能解决的多尺度特征学习挑战

#### 3. 模型架构设计
```
输入数据 → 多模态编码器 → 多尺度特征提取 → 下游任务适配器
     ↓           ↓              ↓              ↓
```

1. **光谱传感器限制**：现有模型大多针对固定光谱传感器设计
2. **仅关注地表**：忽略了大气层等其他重要信息
3. **元数据利用不足**：未能充分利用图像之外的宝贵元数据信息
4. **可扩展性差**：难以适应不断增长的卫星数据规模

Copernicus-FM 提出了**下一代EO基础模型的三大关键组件**：

#### 1. Copernicus-Pretrain：大规模预训练数据集
```
数据规模：18.7M 对齐图像
覆盖范围：所有主要哥白尼哨兵任务
时间跨度：多年数据积累
空间范围：从地球表面到大气层
```

| 技术 | 功能 |
|------|------|

#### 3. Copernicus-Bench：系统评估基准
