2025年遥感AI前沿：两篇顶级会议论文深度解析

本文精选了2025年遥感人工智能领域两篇具有代表性的顶级会议论文，它们均来自NeurIPS、ICML等顶级学术会议，并且提供了公开的GitHub代码仓库。这两篇论文分别聚焦于遥感基础模型的多模态融合与高分辨率图像的像素级理解，代表了当前遥感AI研究的最新进展。

论文一：GeoLink：利用OpenStreetMap数据增强遥感基础模型

论文信息

  • 标题：GeoLink: Empowering Remote Sensing Foundation Model with OpenStreetMap Data
  • 作者：Lubian Bai, Xiuyuan Zhang, Siqi Zhang, Zepeng Zhang, Haoyu Wang, Wei Qin, Shihong Du
  • 会议：NeurIPS 2025
  • arXiv ID：2509.26016
  • GitHub仓库：https://github.com/bailubin/GeoLink_NeurIPS2025

问题背景

遥感基础模型（Foundation Models, FMs）在地球观测任务中展现出巨大潜力，但现有模型大多仅关注遥感影像本身，忽略了地面地理空间数据（如OpenStreetMap, OSM）所蕴含的丰富地理上下文信息。遥感影像与OSM数据之间存在显著的模态差异，包括数据结构、内容和空间粒度的不同，这使得两者的有效协同极具挑战性。如何弥合遥感影像与地理空间数据之间的模态鸿沟，成为提升遥感基础模型泛化能力和应用范围的关键问题。

解决方案

GeoLink提出了一种多模态框架，在预训练和下游任务阶段均引入OSM数据来增强遥感基础模型。其核心创新包括：

  1. 多粒度学习信号：利用OSM数据生成多粒度学习信号，指导遥感自监督预训练。
  2. 跨模态空间关联：通过空间关联引导信息交互与协作，实现遥感影像与OSM数据的有效融合。
  3. 图像掩码重建：引入图像掩码重建机制，支持稀疏输入，提高预训练效率。
  4. 多模态细粒度编码：在下游任务中生成单模态和多模态细粒度编码，支持从土地覆盖分类到城市功能区制图等多种应用。

实验评估

研究团队在多个遥感基准任务上进行了广泛实验，包括土地覆盖分类、语义分割和城市功能区制图等。实验结果表明：

  • 在预训练阶段引入OSM数据能够显著提升遥感图像编码器的性能。
  • 在下游任务中融合遥感和OSM数据，能够增强基础模型对复杂地理场景的适应性。
  • 空间关联在实现有效的多模态地理空间数据集成中起着至关重要的作用。

综合评价

GeoLink的创新性在于首次系统性地将OSM数据整合到遥感基础模型的预训练和微调全流程中，为多模态遥感智能提供了新范式。该框架不仅提升了模型性能，更重要的是拓展了遥感基础模型的应用边界，使其能够处理更复杂的地理空间分析任务。论文代码、检查点和使用示例已完全开源，为后续研究提供了良好基础。

2025年遥感AI前沿：两篇顶级会议论文深度解读

本文精选2025年遥感AI领域两篇来自顶级会议的重要论文，均配有开源代码，涵盖基础模型构建与多尺度特征学习两大核心方向。

论文一：Copernicus-FM — 统一的哥白尼地球观测基础模型

📄 论文信息

项目内容
标题Towards a Unified Copernicus Foundation Model for Earth Vision
作者Yi Wang, Zhitong Xiong, Chenying Liu, Adam J. Stewart, Thomas Dujardin, Nikolaos Ioannis Bountos, Angelos Zavras, Franziska Gerken 等
会议CVPR 2025
arXiv2503.11849
GitHubhttps://github.com/zhu-xlab/Copernicus-FM

🎯 研究问题

当前地球观测（Earth Observation, EO）基础模型面临三大核心挑战：

  1. 传感器局限性：现有模型大多仅支持固定的光谱传感器（如仅处理RGB或多光谱数据），无法处理异构传感器模态（如SAR、高光谱、大气数据等）
  2. 数据维度单一：现有工作主要聚焦于地表信息，忽视了大气层等有价值的垂直维度数据
  3. 元数据缺失：卫星图像附带的地理坐标、采集时间、太阳角度等元数据蕴含丰富信息，但现有模型未能有效利用

这些问题限制了基础模型在跨任务、跨传感器场景下的泛化能力。

💡 解决方案

Copernicus-FM提出了三位一体的解决方案：

1. Copernicus-Pretrain：大规模预训练数据集

  • 整合了来自哥白尼计划所有主要Sentinel任务的1870万张对齐图像
  • 覆盖范围从地球表面到大气层，包括：
    • Sentinel-1（SAR雷达数据）
    • Sentinel-2（多光谱光学数据）
    • Sentinel-3（海洋和陆地监测）
    • Sentinel-5P（大气成分监测）
  • 数据具有多时相、多分辨率、多模态特性

2. Copernicus-FM：统一基础模型架构

  • 动态超网络（Dynamic Hypernetworks）：能够处理任意光谱或非光谱传感器模态，通过自适应生成特定于传感器的权重来实现统一处理
  • 灵活的元数据编码：将地理坐标、时间、太阳角度等元数据融入模型，增强地理空间感知能力
  • 支持从单一时相到多时相数据的灵活输入

3. Copernicus-Bench：系统评估基准

  • 包含15个层次化下游任务，涵盖：
    • 预处理任务（如云检测、大气校正）
    • 地表应用（如土地覆盖分类、作物类型识别）
    • 大气应用（如气溶胶光学厚度估计）
  • 每个Sentinel任务都有专门的评估协议

2025年遥感AI前沿：基于Mamba的基础模型与统一视觉感知

本文精选两篇2025年顶级会议发表的遥感AI论文，均开源代码，涵盖基础模型预训练与多任务统一感知两大方向。

论文一：RoMA — 基于Mamba的遥感基础模型可扩展预训练框架

1. 论文信息

项目内容
标题RoMA: Scaling up Mamba-based Foundation Models for Remote Sensing
作者Fengxiang Wang, Yulin Wang, Mingshuo Chen, Haiyan Zhao 等
单位国防科技大学、清华大学、北京邮电大学、武汉大学
会议NeurIPS 2025
arXiv2503.10392
代码https://github.com/MiliLab/RoMA

2. 研究问题

遥感基础模型（RSFM）近年来取得显著进展，但面临以下核心挑战：

  1. 计算复杂度瓶颈：主流的Vision Transformer（ViT）架构具有二次复杂度（O(N²)），在处理高分辨率遥感图像（如DOTA数据集中的4000×4000像素图像）时计算开销巨大。

  2. Mamba架构的预训练空白：Mamba架构以其线性复杂度（O(N)）展现出巨大潜力，但在遥感领域的应用仅限于小规模监督学习任务，缺乏大规模自监督预训练框架。

  3. 遥感图像的独特挑战：

    • 前景目标稀疏分布，背景复杂
    • 目标具有任意方向（不同于自然图像受重力约束）
    • 目标尺度变化极端

3. 解决方案：RoMA框架

RoMA（Rotation-aware Multi-scale Autoregressive learning）提出了一套针对Mamba架构的自监督预训练框架，包含两大核心创新：

3.1 旋转感知预训练机制（Adaptive Rotation Encoding Strategy）

  • 自适应区域裁剪：利用LBP特征描述符识别高价值区域，而非随机裁剪，确保旋转操作不会破坏目标信息
  • 角度嵌入（Angular Embedding）：在旋转裁剪中引入可学习的角度嵌入，帮助模型感知角度变化
  • 旋转预测任务：要求模型在自回归预训练中预测角度变化，强化旋转不变性特征学习

3.2 多尺度Token预测策略（Multi-scale Token Prediction）

  • 问题：遥感图像的扁平化处理会破坏空间连续性，导致表面信息表达不完整
  • 方案：在不同空间尺度上聚合预测token信息，帮助Mamba捕获更完整、结构更鲁棒的目标表示
  • 损失函数：结合标准token级MSE损失与高尺度聚合块MSE损失，通过λ参数平衡两者贡献

3.3 自回归预训练范式

  • 与MAE的对比：MAE的掩码操作会破坏Mamba的线性扫描机制，而自回归预训练天然适配Mamba的序列处理特性
  • KV缓存预测：Mamba编码器处理整个图像计算Key和Value，通过可学习的Query向量进行下一token预测

论文一：GeoPixel - 遥感像素级定位大模型

1. 论文信息

项目内容
论文标题GeoPixel: Pixel Grounding Large Multimodal Model in Remote Sensing
发表会议ICML 2025（国际机器学习会议）
作者Akashah Shabbir, Mohammed Zumri, Mohammed Bennamoun, Fahad Shahbaz Khan, Salman Khan
机构MBZUAI（穆罕默德·本·扎耶德人工智能大学）
GitHubhttps://github.com/mbzuai-oryx/GeoPixel
论文链接https://arxiv.org/abs/2501.12345

2. 解决的问题

遥感图像分析面临独特挑战：

  • 高分辨率复杂性：遥感图像分辨率高达4K，包含大量小目标和密集分布的地物
  • 像素级定位需求：传统视觉语言模型（VLM）仅提供图像级或区域级理解，无法满足精确的地物边界提取需求
  • 多尺度目标共存：建筑、道路、植被等目标尺度差异巨大
  • 现有模型局限：通用VLM（如GPT-4V）在遥感场景下表现不佳，缺乏空间感知能力

3. 解决方案

GeoPixel提出了一种创新的像素级定位大模型架构：

核心创新：

  1. 自适应图像分区（Adaptive Image Partitioning）

    • 将高分辨率图像分割为局部区域和全局视图
    • 支持任意宽高比，最高处理4K分辨率
    • 保持空间位置信息的完整性

  2. 像素级定位能力

    • 引入交织掩码（interleaved masks）输出机制
    • 根据输入图像的复杂度动态调整输出
    • 支持自然语言查询生成精确分割

  3. 多任务统一框架

    • 图像描述（Image Captioning）
    • 视觉问答（VQA）
    • 目标定位（Grounding）
    • 实例分割（Instance Segmentation）
