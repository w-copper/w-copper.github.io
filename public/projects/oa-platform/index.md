# 📋 OA 公文管理系统

支持钉钉单点登录、OnlyOffice 在线编辑的公文管理系统

## ✨ 核心功能

### 用户功能
- 📄 **公文管理**：创建、编辑、审批公文
- 📝 **在线编辑**：集成 OnlyOffice 在线文档编辑
- 🔐 **钉钉登录**：钉钉 SSO 单点登录
- 📊 **审批流程**：多级审批流程管理
- 📥 **文件下载**：导出公文和审批单

### 管理功能
- 👥 **用户管理**：用户增删改查
- 📋 **角色权限**：多角色权限控制
- 📝 **操作日志**：完整的操作记录
- 💾 **数据备份**：一键备份和恢复

## 🛠️ 技术栈

| 层级 | 技术 |
|------|------|
| **前端** | Vue 2 + Element UI |
| **后端 1** | Django (主服务) |
| **后端 2** | Spring Boot (文档生成) |
| **数据库** | MySQL |
| **文档编辑** | OnlyOffice |
| **反向代理** | Nginx |
| **部署** | Docker Compose |

## 🚀 快速启动

```bash
# 克隆项目
cd D:\project\oa-platform

# 启动 Docker 服务
cd deploy
docker-compose up -d

# 访问系统
http://localhost/archive/login
```
