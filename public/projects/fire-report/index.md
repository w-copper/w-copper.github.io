# 🔥 火点监测平台

基于 React + Cesium + FastAPI 的火点监测平台，支持 NASA FIRMS 数据、AI 报表生成、多省份登录管理。

## ✨ 核心功能

### 普通用户功能
- 🗺️ **3D 地图展示**：基于 Cesium 的三维地球，实时展示火点信息
- 📊 **多数据源支持**：NASA FIRMS 火点数据 + 自定义火点数据
- ⏱️ **时间筛选**：快捷选择最近 10 分钟、24 小时、7 天的火点
- 📈 **时间轴**：拖动时间轴查看历史火点变化
- 🤖 **AI 报表**：基于 LLM 的智能火点分析报表
- 💬 **AI 对话**：与 AI 交互，问答式生成报表
- 📥 **数据下载**：导出火点数据和 AI 报表

### 管理员功能
- 👥 **多省份管理**：分省登录，数据隔离
- ⚙️ **数据源配置**：管理 WFS 数据源（URL、Map Key、时间范围）
- 🤖 **AI Provider 配置**：支持 OpenAI 及兼容接口
- 📝 **操作日志**：完整的用户操作记录

## 🛠️ 技术栈

| 层级 | 技术 |
|------|------|
| **前端** | React + Cesium + Tailwind CSS + Vite |
| **后端** | FastAPI + SQLAlchemy |
| **数据库** | PostgreSQL + PostGIS |
| **缓存** | Redis |
| **任务队列** | Celery |
| **部署** | Docker Compose |
